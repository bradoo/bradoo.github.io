## 欢迎使用博客生成器

这是一篇示例文章，展示了如何使用 Markdown 格式编写博客文章。

### 基本语法

#### 标题

使用 `#` 符号创建标题，`#` 的数量表示标题级别。

#### 段落

直接书写文字即可创建段落。段落之间用空行分隔。

#### 强调

- **粗体文字**：使用 `**文字**` 或 `__文字__`
- *斜体文字*：使用 `*文字*` 或 `_文字_`
- ***粗斜体***：使用 `***文字***`

### 代码

#### 行内代码

#### 代码块

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}

greet('World');
```

```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```
