# 📝 可视化编辑器使用指南

## 🎯 编辑器简介

这是一个功能强大的所见即所得（WYSIWYG）博客文章编辑器，让您可以在浏览器中直接编写和预览文章，无需手动编写 Markdown 语法。

### ✨ 主要特性

- 🎨 **所见即所得**: 左侧编辑，右侧实时预览
- 📝 **Markdown 支持**: 完整的 Markdown 语法支持
- 💾 **自动保存**: 编辑内容自动保存到浏览器
- ⌨️ **快捷键**: 支持常用快捷键操作
- 📊 **字符统计**: 实时显示字符数和阅读时间
- 🎯 **表单验证**: 确保必填项完整
- 📥 **一键下载**: 直接下载为 Markdown 文件

## 🚀 快速开始

### 1. 打开编辑器

```bash
# 方式一：命令行打开
open blog-generator/editor.html

# 方式二：浏览器打开
# 在浏览器中打开文件：blog-generator/editor.html

# 方式三：本地服务器（推荐）
cd bradoo.github.io
python3 -m http.server 8000
# 访问 http://localhost:8000/blog-generator/editor.html
```

### 2. 填写文章信息

#### 必填项
- **文章标题**: 简洁明了的标题（最多100字符）
- **发布日期**: 选择文章发布日期
- **分类**: 选择文章所属分类
- **文章内容**: 使用 Markdown 编写正文

#### 可选项
- **标签**: 多个标签用逗号分隔
- **文章简介**: 用于 SEO 和摘要（最多200字符）
- **封面图片**: 图片路径，如 `/images/cover.jpg`
- **阅读量**: 初始阅读量

### 3. 编写内容

在 Markdown 编辑器中编写文章内容，支持以下功能：

#### 工具栏功能
- **B** - 粗体
- **I** - 斜体
- **H** - 标题
- **"** - 引用
- **•** - 无序列表
- **1.** - 有序列表
- **🔗** - 插入链接
- **🖼️** - 插入图片
- **📊** - 插入表格
- **👁️** - 预览模式
- **⚡** - 分屏模式
- **⛶** - 全屏模式

#### Markdown 语法快速参考

```markdown
# 一级标题
## 二级标题
### 三级标题

**粗体文字**
*斜体文字*
***粗斜体***

[链接文字](https://example.com)
![图片描述](/images/photo.jpg)

- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2

> 这是引用文字

`行内代码`

​```javascript
// 代码块
console.log('Hello World');
​```

| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |
```

### 4. 实时预览

右侧预览面板会实时显示文章的最终效果，包括：
- 文章标题和元信息
- 格式化的正文内容
- 预计阅读时间
- 完整的样式效果

### 5. 保存文章

点击"💾 保存为 Markdown"按钮：
1. 自动生成包含 Front Matter 的 Markdown 文件
2. 文件会自动下载到您的下载目录
3. 将文件移动到 `blog-generator/content/` 目录
4. 运行构建脚本生成 HTML

## ⌨️ 快捷键

### 编辑器快捷键
- `Ctrl/Cmd + B` - 粗体
- `Ctrl/Cmd + I` - 斜体
- `Ctrl/Cmd + K` - 插入链接
- `Ctrl/Cmd + Alt + I` - 插入图片
- `Ctrl/Cmd + '` - 引用
- `Ctrl/Cmd + L` - 无序列表
- `Ctrl/Cmd + Alt + L` - 有序列表
- `Ctrl/Cmd + Alt + C` - 代码块
- `F9` - 预览模式
- `F11` - 全屏模式

### 全局快捷键
- `Ctrl/Cmd + S` - 保存文章

## 🎨 界面说明

### 左侧编辑面板
- **文章信息表单**: 填写标题、日期、分类等
- **Markdown 编辑器**: 编写文章正文
- **工具栏**: 快速插入 Markdown 语法
- **状态栏**: 显示行数、字数、光标位置

### 右侧预览面板
- **文章头部**: 显示分类、标题、元信息
- **文章正文**: 实时渲染的 HTML 内容

### 底部操作栏
- **🗑️ 清空**: 清空所有内容（需确认）
- **👁️ 预览**: 滚动到预览面板
- **💾 保存**: 下载 Markdown 文件

## 💡 使用技巧

### 1. 自动保存
编辑器会自动保存您的内容到浏览器本地存储，即使关闭页面也不会丢失。

### 2. 图片处理
```markdown

![描述](/images/photo.jpg)

# 外部图片
![描述](https://example.com/image.jpg)

# 带链接的图片
[![描述](/images/photo.jpg)](https://example.com)
```

### 3. 代码高亮
```markdown

```markdown

```

## 🔧 常见问题

### Q: 编辑器打不开？
A: 确保使用现代浏览器（Chrome、Firefox、Safari、Edge），并且启用了 JavaScript。

### Q: 内容丢失了？
A: 编辑器有自动保存功能，刷新页面后内容会自动恢复。

### Q: 如何插入本地图片？
A: 先将图片放到 `/images/` 目录，然后使用相对路径引用：`![描述](/images/photo.jpg)`

### Q: 预览和实际效果不一致？
A: 预览使用的是简化样式，实际效果以生成的 HTML 页面为准。

### Q: 如何批量编辑文章？
A: 可以直接编辑 `blog-generator/content/` 目录下的 Markdown 文件。

### Q: 支持哪些 Markdown 扩展？
A: 支持 GFM（GitHub Flavored Markdown），包括表格、任务列表、删除线等。

## 📊 工作流程

### 完整流程

1. **打开编辑器**
   ```bash
   open blog-generator/editor.html
   ```

2. **编写文章**
   - 填写文章信息
   - 编写正文内容
   - 实时查看预览

3. **保存文章**
   - 点击"保存为 Markdown"
   - 文件自动下载

4. **移动文件**
   ```bash
   mv ~/Downloads/article-name.md blog-generator/content/
   ```

5. **生成 HTML**
   ```bash
   ./blog-generator/build.sh
   ```

6. **部署上线**
   - 脚本会自动提示是否部署
   - 或手动 `git push`
